# Repositorio de Pruebas

## Descripción

Este repositorio está dedicado a **probar flujos de trabajo con Pull Requests y Draft PRs** en GitHub. Aquí experimentamos con:

- Creación de ramas y pull requests
- Draft pull requests
- Revisión de código
- Procesos de merge y colaboración

## Alcance

📋 **Propósito principal:** Laboratorio para probar y entender los mecanismos de PR en GitHub.

### No es un proyecto en producción
Este repo sirve únicamente como espacio de experimentación y pruebas.
